This Excel practice project demonstrates how to parse and clean text data by identifying space positions and splitting full names into First, Middle, and Last names.

## Problem Statement

The dataset contains an **Agent Name** column with full names in different formats, including:
- Two-word names (First + Last)
- Three-word names (First + Middle + Last)
- Multi-word names

The objectives were:

**Q1:** Split full names into  
- First Name  
- Middle Name  
- Last Name  

**Q2:** Identify  
- First space position  
- Second space position  
- Third space position
